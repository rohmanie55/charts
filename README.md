## 🚀 Installation

To install the charts repository:

```bash
$ helm repo add soketi https://helm.soketi.app
$ helm repo update
```

To install a specific chart:

```bash
$ helm install some-release soketi/<chart>
```

## Search for available repos

```bash
$ helm search repo
```
